User is developing "Space Bandits", a Space Invaders-style game for the Tandy Color Computer 3 (CoCo 3). They are familiar with the CoCo 3 hardware constraints (1.8MHz 6809/6309 CPU, 512KB banked RAM, 320x200 16-color graphics). The game uses a modified DynoSprite engine. There is also a macOS port for testing purposes.
